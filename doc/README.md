## Getting Started

**New to the project?** Start here:
1. Read [CONTRIBUTING.md](CONTRIBUTING.md) for setup instructions
2. Review [MYST_CONVENTIONS.md](MYST_CONVENTIONS.md) before editing content
3. Use [MAINTENANCE.md](MAINTENANCE.md) as your command reference

**Working with scripts?**
- See [scripts/README.md](scripts/README.md) for available utilities
- Run `npm run --list` to see all available npm commands

**Building exports (PDF/DOCX)?**
- See [LATEX_TO_TYPST_MIGRATION.md](LATEX_TO_TYPST_MIGRATION.md) for export configuration
